# Screenshots Directory

Place your application screenshots here. The README.md references these images:

- `dashboard-overview.png` - Main dashboard view
- `asset-selection.png` - Asset selector component
- `price-dashboard.png` - Price dashboard with chart
- `ai-prediction.png` - AI prediction card
- `key-factors.png` - Key factors analysis section
- `backtesting-results.png` - Backtesting results display
- `detailed-metrics.png` - Detailed metrics breakdown
- `ai-analysis.png` - AI analysis text section

## Image Requirements

- **Format**: PNG or JPG
- **Recommended Size**: 1920x1080 or larger
- **Quality**: High resolution for clarity
- **Naming**: Use the exact filenames listed above

## Adding Screenshots

1. Take screenshots of your application
2. Save them with the exact filenames listed above
3. Place them in this directory
4. The README will automatically reference them
